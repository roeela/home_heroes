# HomeHeroes — ניהול תורנויות ביתיות

Flutter + Firebase Android app for a single family to manage home chores.
All UI is in **Hebrew with RTL layout** (Rubik font).

---

## Quick start

### Prerequisites
- Flutter SDK installed
- Android SDK installed
- A Firebase project (see Firebase Setup below)

### 1. Install dependencies
```
flutter pub get
```

### 2. Configure Firebase (one-time)
See **Firebase Setup** section below.

### 3. Run
```
flutter run
```

## Firebase Setup

> Until this is done the app crashes on startup with an `UnsupportedError` from `firebase_options.dart`.

### Step-by-step

1. **Create a Firebase project** at https://console.firebase.google.com

2. **Enable services** in the Firebase Console:
   - Authentication → Sign-in method → Google (enable it)
   - Firestore Database → Create database (start in **test mode** for development)

3. **Install FlutterFire CLI** (once per machine):
   ```
   dart pub global activate flutterfire_cli
   ```

4. **Configure** from inside the `home_heroes` directory:
   ```
   flutterfire configure
   ```
   - Select your Firebase project
   - Choose Android only
   - This overwrites `lib/firebase_options.dart` and `android/app/google-services.json`

5. **Add your debug SHA-1** to Firebase Console → Project Settings → Your Android App → Add fingerprint:
   ```
   keytool -list -v -keystore %USERPROFILE%\.android\debug.keystore -alias androiddebugkey -storepass android -keypass android
   ```
   Copy the `SHA1:` line and paste it into Firebase Console.

6. Run `flutter run` — the app should launch and Google Sign-In should work.

### Firestore security rules (after dev)

Replace the default permissive rules in Firebase Console → Firestore → Rules:
```
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /family/{familyId}/{document=**} {
      allow read, write: if request.auth != null
        && exists(/databases/$(db)/documents/family/$(familyId)/users/$(request.auth.token.email));
    }
  }
}
```

## First-time app flow

1. **Primary parent** opens app → signs in with Google
2. No family found → taps "צור משפחה חדשה" → enters family name → family is created
3. Parent goes to **Family tab** → "+" → enters each child's Gmail + their weekly quota
4. Each child opens app → signs in with Google → automatically joins the family
